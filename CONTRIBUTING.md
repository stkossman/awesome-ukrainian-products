This is a curated list of awesome software, services, and products made in Ukraine or by Ukrainians.

## How to Contribute

1. **Fork the repository** and create a branch for your addition.
2. **Check for duplicates**. Ensure the item isn't already in the list.
3. **Add your item** to the appropriate category. If unsure, check existing items or create a new category if it makes sense.
4. **Format your entry**. Use the following format:
   ```markdown
   - [Item Name](https://link-to-item.com) - A short, objective description ending with a dot. (Badges here)
5. **Use Badges**. Please use the appropriate text badges (see README Legend) to categorize the item.
6. **Commit and Push**. Make sure your commit message is clear (e.g., `feat: add Grammarly`).
7. **Open a Pull Request**.

## Quality Standards

* **Relevance**: The product must be of Ukrainian origin (founded/developed in Ukraine or by Ukrainians).
* **Functionality**: The product must be usable (no "coming soon" landing pages).
* **Description**: Keep it short, informative, and neutral. Avoid marketing fluff like "The best ever...".
